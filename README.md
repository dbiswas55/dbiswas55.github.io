# GitHub Pages Portfolio Package

This package contains a simple static site you can upload directly to a GitHub repository and publish with GitHub Pages.

## Files included
- `index.html` — main page
- `styles.css` — site styling
- `script.js` — mobile navigation and footer year

## How to upload
1. Create or open the repository you want to use for GitHub Pages.
2. Upload these files to the root of the repository.
3. Commit the changes.
4. In GitHub, go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or your active branch)
   - **Folder:** `/root`
6. Save the settings.
7. GitHub Pages will publish the site at your repository page URL.

## Recommended usage
- For a personal site, use a repository named exactly:
  `dbiswas55.github.io`
- For a project page, you can use any repository name.
