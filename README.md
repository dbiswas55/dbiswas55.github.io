# Dipayan GitHub Pages Starter

This folder contains a simple, no-build personal website:

- `index.html` (your whole site)
- `style.css` (minimal styling)
- `assets/Dipayan_Biswas_CV.pdf` (your CV PDF)

## Quick publish to GitHub Pages

1. Create a repo named `dbiswas55.github.io` (public).
2. Upload these files to the repo root.
3. Go to **Settings → Pages**:
   - Source: **Deploy from a branch**
   - Branch: `main` (root)
4. Your site will be at: `https://dbiswas55.github.io/`

## Add your photo

Add a square-ish photo as `assets/profile.png` (or change the filename in `index.html`).

## Edit content

Open `index.html` and search/replace the text in each section.
